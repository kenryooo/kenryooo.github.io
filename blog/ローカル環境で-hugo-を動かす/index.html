ローカル環境で Hugo を動かす
Jan 5, 2017
2 minutes read

Hugo とは

Go 製の静的サイトジェネレータ。
静的ページの生成が高速で、シンプルなところが好み。

インストール

mac なら

$ brew install hugo

Go 環境が設定済みなら下記でもいい

$ go get github.com/spf13/hugo

プロジェクト作成

$ hugo new site <your_site>
$ cd <your_site> # ここがプロジェクトのルートディレクトリになる

テーマの適用

<your_site> 配下の themes というディレクトリ配下に配置する。

すべてのテーマをまとめてインストールする場合

$ git clone --depth 1 --recursive https://github.com/spf13/hugoThemes.git themes

テーマを個別に選んでからインストールする場合

こちらからテーマを選んで、個別のリポジトリから git clone する。ここでは cocoa を例に進める。

$ git clone https://github.com/nishanths/cocoa-hugo-theme.git themes/cocoa
$ hugo -vw -t cocoa serve  # サーバを起動 http://localhost:1313/)

Theme のカスタマイズ

カスタマイズするには <your_site>/themes/<theme>/ 配下を直接編集せずに、 直下にディレクトリ・ファイルを追加する。追加するファイルは <your_site>/themes/<theme>/exampleSite からコピーする。

まず exampleSite の config.toml をコピーして、カスタマイズしていくと楽ちん。あとは、Configuring Hugoのページを見ながらカスタマイズしていくとよい。
